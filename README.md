# China Journey Kit

Static travel guide site for international visitors planning China trips.

## Local preview

Open `index.html` directly or run:

```bash
python3 -m http.server 4173
```

## Deployment

- GitHub Pages: publish this directory as static files.
- Netlify: deploy the folder; `_headers` and `netlify.toml` provide security headers.
- Vercel: deploy the folder; `vercel.json` provides security headers.

Before paid traffic, replace local demo forms with a real backend or form service, add analytics consent, update legal company details, and configure a production domain.
